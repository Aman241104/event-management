# Project Overview

The **Event Manager (Prestige Edition)** is a high-fidelity, luxury SaaS platform designed for a premium event management agency. It focuses on **minimalist opulence**, **emotional storytelling** via cinematic galleries, and a **"Mobile-First Concierge"** experience powered by the WhatsApp Business API.

## Design Philosophy
- **Minimalist Opulence:** Generous whitespace, no UI noise, and 1px Gold borders.
- **Emerald & Obsidian Palette:** Timeless luxury theme (Deep Forest, Obsidian, Gold, Ivory).
- **Editorial Typography:** Playfair Display (Serif) paired with Inter (Sans).
- **Sophisticated Interaction:** Slow, intentional GSAP animations (1.2s - 1.8s).
- **Performance is Luxury:** Sub-second loads and zero layout shifts via Next/Image optimization.
