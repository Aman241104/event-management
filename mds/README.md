# Documentation Directory

This directory contains the core documentation for the Zing Bliss Events platform, including design principles, visual systems, and implementation guides.

## Files
- `design-principles.md`: High-level creative direction.
- `visual-design.md`: Technical design specifications (colors, typography, components).
- `implementation.md`: Technical implementation roadmap.
- `plan.md`: Current project status and progress tracking.
- `dev-implementation.md`: Specific developer tasks for the current sprint.
