# Visual Design System (The Ivory & Verdant Collection)

This document provides a highly detailed specification for the "Light Luxury" visual identity.

## Color System

The system uses a Warm Paper base with Heritage Green and Burnished Gold accents.

- **Background Colors:** 
  - Main Canvas: Bone Paper (`#FCFBF7`)
  - Sub-Background: Dusty Ivory (`#F4F1EA`)
- **Primary Accent:** Heritage Green (`#2A4D37`)
- **Secondary Accent:** Burnished Gold (`#C5A059`)
- **Text Colors:**
  - Primary Text: Ebonized Wood (`#1C1C1C`)
  - Secondary Text: Stone Gray (`#4A4A4A`)
  - Muted/Disabled: Linen Gray (`#AFAFAF`)
- **Border Colors:**
  - Default Border: Linen (`#D9D4C7`)
  - Prestige Border: Burnished Gold (`#C5A059`)
- **Hover States:** 
  - Primary Hover: Deep Forest (`#1B3325`)
  - Surface Hover: Soft Sand (`#E8E4D9`)

## Typography System

An editorial pairing of Serif and Sans-Serif fonts.

- **Primary Heading (Serif):** `Playfair Display`
- **Body & UI (Sans):** `Inter`
- **Metadata (Mono):** `JetBrains Mono`

**Heading Hierarchy:**
- **H1 (Hero):** Desktop: 72px | Weight: 700 | Serif | Line Height: 1.1
- **H2 (Section):** Desktop: 48px | Weight: 600 | Serif | Line Height: 1.2
- **H3 (Card):** Desktop: 28px | Weight: 500 | Serif | Line Height: 1.3

## Component Design

### Buttons
- **Primary:** Background Heritage Green (`#2A4D37`), Text White. Sharp 2px radius.
- **Secondary:** Transparent background, Border 1px Burnished Gold, Text Heritage Green.
- **Hover:** Primary shifts to Deep Forest; Secondary fills with Soft Sand.

### Cards
- **Visuals:** Background Dusty Ivory (`#F4F1EA`), Border 1px Linen (`#D9D4C7`).
- **Hover:** 1px Burnished Gold border reveal.

### Navigation Bars
- **Visuals:** Glassmorphic Ivory (backdrop-filter blur 12px, background `rgba(252, 251, 247, 0.8)`).
- **Border:** Subtle 1px Linen on the bottom edge.
